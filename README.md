Void-Nuke’s interface is built for ease of use. Here are common tasks and how to do them:

- Find the "Tokens" tab or option.
- Paste your Discord tokens into the provided box.

Tokens allow the program to access multiple Discord accounts automatically.

### Join Servers Automatically

- Go to the "Join" section.

### Multi-Tool Options

## 🛠️ Troubleshooting Common Issues

- **Program Won’t Launch**: Make sure you have installed on a compatible Windows machine. Try running as Administrator.

- **Tokens Not Working**: Check that tokens are valid and not expired. Invalid tokens won’t be accepted.

- **Join Server Fails**: Verify the server invite link is still active.

- **Slow Performance**: Close other heavy applications and ensure you have a reliable internet connection.
